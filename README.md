# storybook

storybook    storybook ui视图

# 1 准备运行环境
```
npm install
```

----------------------------------------------------------------------

# 2 启动react组件文档
```
npm run storybook-react
```

----------------------------------------------------------------------

# 3 启动vue组件文档
```
npm run storybook-vue
```

----------------------------------------------------------------------

# 4 打包成静态网页
根目录上执行命令即可，会一起打包三个目录下文件
```
npm run build
```
